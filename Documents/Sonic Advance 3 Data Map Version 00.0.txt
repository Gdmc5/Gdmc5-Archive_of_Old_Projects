Text Format

0151 Spacer Mini Note The next 2 bytes tell the size of the space
0152 Timer Mini Note The next 2 bytes tell the speed which letters appear
0153 Line break Mini Note like when you press ENTER in a text editor
0154 Button request Mini Note Player must press A button to see the rest of the text
0155 EOT Mini Note (end of text)
